Ниже представлены все страницы, которые я сделал, также, описание к ним
Интерактивный обратный остчёт
Описание
Обратный отсчёт, где для начала необходимо кликать определенное колличество раз, после чего появится окна для ввода числа, с которого будет начат отсчёт.
Генератор записей
Описание
Страница представляет собой генератор записей. Вверху есть кнопка, после нажатия которой появится окно, где нужно будет указать заголовок и описание записи, после её создания можно будет выбрать картинку в ней или удалить запись, если не понравилась.
Калькулятор
Описание
Страница является обычным калькулятором, но с неготовым функционалом. По сути, это просто пустая оболочка, поэтому посещать её не рекомендуется.
Простой обратный остчёт
Описание
Очень простой обратный остчёт, где для запуска необходимо в появляющемся окне просто указать, с какого числа начинать отсчёт.
Интерактивные картинки
Описание
По центру страницы есть несколько картинок, которые по нажатию кнопок можно переместить в сайдбар (поле слева на странице). В сайдбаре есть кнопка, которая может вывести в консоль браузера колличество картинок, находящихся в ней.
Генератор паролей
Описание
Это обычный генератор пароля, который сездает только числовой пароль. В появляющемся окне цифрой можно указать, какой длинны будет пароль и затем кнопкой сгенерировать его.
Скучные интерактивные картинки
Описание
В сайдбаре (поле слева на странице) есть несколько картинок, которые нажатием можно переместить на основную страницу, затем, двойным нажатием обратно в сайдбар. Страница так называется, потому что на самом деле является скучной по функционалу.
Сфетофор
Описание
Проект представляет собой сделанный по рофлу светофор, с цветами которого можно поиграться. Можно запустить или остановить автомобиль, больше в нём ничего нельзя сделать.
Тройные картинки
Описание
Это сменяющие друг друга раз в 5 секунд картинки, которые послужили основой первого блока на этой станице. За картинками можно как наблюдать, так и вручную сменять.
